# Zod-Com-Node

Projeto de estudo, onde aprendo como utilizar o Zod com Node.

Passo a passo:

1) Rodei o comando npm i -D @types/node tsx typescript para instalar as dependencias basicas;
2) Rodei o comando npm install zod, para instalar o zod no projeto;
3) Criei o arquivo zod.ts que utilizarei para a configuração da lib;
4) Personalizei o script "zod": "node --import=tsx --watch ./src/zod.ts", no package.json;
